Welcome To VSCode Extension For webezy.io Framework

Webezy.io is a free open-source framework for creating, managing and deploying gRPC micro-services with ease.

It is doing so with a simple but powerful CLI and by leveraging it into vscode extension for native functionality.

Webezy.io framework lets you define your services and thier functionality once and then use it across multiple different environments and languages, currently we support both for Server side And Client side Python & Typescript

Usage

Install webezy.io CLI

To use this extension and to start creating your gRPC services with vscode extension you will need to download and install the webezy.io CLI
